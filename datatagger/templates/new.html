What is IndicNLP?

IndicNLP is an open source initiative for NLP on Indigenous languages.

Indian languages are complex and bring about unique challenges for NLP practitioners. In IndicNLP, we try to build language models, tools and techniques for NLP on Indian languages and use them to solve NLP tasks. We also try to consolidate and curate all open source work and data in NLP on Indian Languages.There are just so many things to do. And we need your help!

You can’t code for every permutation combination of rules that is possible with a Language. Artificial Intelligence is a way to learn the rules from the data and solve the problem.

Why Open Source?

We all love Linux. Current cloud computing or AI wouldn’t be possible with Linux being open sourced. Most of the regional languages in India doesn’t have the market size to attract investments from big corporations to build the software in a profitable way. Open source initiative is it Kick start to build the tools or software and get the wheels rolling.
